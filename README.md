# DAIDAI | Authentic Filipino Cuisine

A responsive, single-page website for **DAIDAI**, an authentic Filipino kitchenette, grill (ihawan), and sari-sari store located in Aichi, Japan. 

## 🌟 Features
* **Modern UI:** Built with Tailwind CSS for a clean, mobile-first responsive design.
* **Interactive Menu:** Dynamic tabbed navigation for categorizing meals, desserts, and grill items.
* **Quick Access:** Embedded Google Maps links, direct phone calling, and Facebook integration.
* **No-Build Setup:** Runs entirely on static HTML with CDN-based styling and vanilla JavaScript.

## 🛠️ Tech Stack
* **HTML5**
* **Tailwind CSS** (via CDN)
* **Vanilla JavaScript** (Inline)
* **Fonts & Icons:** Google Fonts (Poppins, Noto Sans JP, Dancing Script) & FontAwesome
